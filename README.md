# streams
轻便的gb28181协议中的rtp+ps格式视频流的封装和解析

## packet 
packet实现ps的相关封装和解析,

### example/enc 
通过[joy4](https://github.com/nareix/joy4)来读本地视频文件，然后调用`RtpTransfer`对象进行封装ps。

### example/dec
